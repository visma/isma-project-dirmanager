Preferences

La configuration des preferences permet de sauvegarder les chemins source et cible utilisés

Généralités

Les boutons de la barre d'outils permettent d'interagir avec la plupart des fonctionnalités accesibles via les onglets.



Les 2 boutons expand/collaspe permettent de replier/dérouler le ou les arbre(s)
Le bouton charger permet de charger le ou les arbre(s)
Le bouton lancer permet de lancer le traitement lié à l'onglet actif
