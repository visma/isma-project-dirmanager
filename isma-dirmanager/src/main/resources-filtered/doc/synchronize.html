Copie de la source vers la destination les fichiers manquants au même niveau d'arborescence.
Ne recopie pas les fichiers déjà existants dans l'arborescence de destination les fichiers déjà existants.

Signalétique des couleurs

  • Bleu : le fichier n'existe pas de l'autre côté
  • Vert : le fichier a été copié de ce côté
  • Noir : le fichier a été copié de l'autre côté ou il était déjà synchrone
